hexo博客的搭建

发表于 2018-11-03 | 分类于 hexo

什么是Hexo？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装前提

安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装下列应用程序：

1.Node.js
2.Git

如果您电脑已经安装上述的必备程序，接下来只需要使用node自带的npm包管理工具可完成Hexo的安装。

在window用户安装需要注意Git安装的时候，需要勾选Add to path选项。主要这个选项一勾选，安装的任何命令工具，直接默认安装到系统path路径（在我的电脑属性的环境变量里面），使用git Bash的工具进行安装，提供了Linux风格的shell，在该环境下，可以直接用来安装node.js。

安装Hexo

建站

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

新建完成后，指定文件夹的目录如下：

_config.yml

网站的配置信息,您可以在此配置大部分的参数

package.json

应用程序的信息.

scaffolds

模版 文件夹。当您新建文章时，Hexo 会根据 scaffold 来建立文件。

Hexo的模板是指在新建的markdown文件中默认填充的内容。例如，如果您修改scaffold/post.md中的Front-matter内容，那么每次新建一篇文章时都会包含这个修改(hexo new [layout] title)

source

资源文件夹是存放用户资源的地方。除 _posts 文件夹之外，开头命名为 _ (下划线)的文件 / 文件夹和隐藏的文件将会被忽略。Markdown 和 HTML 文件会被解析并放到 public 文件夹，而其他文件会被拷贝过去。

themes

主题 文件夹。Hexo 会根据主题来生成静态页面。

配置

您可以在 _config.yml 中修改大部份的配置。

网站

参数 描述
title 网站标题
subtitle 网站副标题
description 网站描述
author 您的名字
language 网站使用的语言
timezone 网站时区。Hexo 默认使用您电脑的时区。时区列表。比如说：America/New_York, Japan, 和 UTC 。

其中，description主要用于SEO，告诉搜索引擎一个关于您站点的简单描述，通常建议在其中包含您网站的关键词。author参数用于主题显示文章的作者。

我的经历

    高中时期,读了一个不起眼的高中学校,很烂,教学方式也是很差,为什么我会读这个学校,要从我初中说起了,我初中那时候年少不懂事,处于叛逆期,老是跟父母做对,父母说二,我就敢说一,因为那时候我迷恋上了一款腾讯游戏”DNF”,那时期老是贪玩,很不喜欢学习,所以才对导致今天这样.在初中时候唯一先接触的一款2D动作游戏就是它,那时候这款游戏在之前是很火的一款游戏,那时候我的同学和小伙伴们基本都是在说这个游戏,上课聊,下课聊,放学聊,无时无刻都在聊,这款”DNF”为什么这么好玩,(这里顺便帮腾讯打一下广告,免费不收广告费) ,”DNF”他主要魅力在于刷粉,爆SS,爆CC,大比例销售拍卖,升级带人刷图,贼爽,通过DNF可以利用搬砖来赚RMB,既可以玩,也可以赚钱的游戏,你说火不火,我之前玩这款游戏,都很少充过人民币,基本都是自己赚的,玩都玩疯了,到了初三要中考的时候,已经来不及,因为成绩不理想而选择这个高中,那时候我还很满怀期待的说到了高中,我要好好读书,考一个好的大学,结果还是凉了.在高中三年里面,不是打篮球,玩电子游戏,就是睡觉,说到这些现在是愧疚也是没用了,毕竟都过去了,不过还是挺开心的,上课有时候逃课和小伙伴去打篮球,打篮球是我高中最迷上一种运动,是我从小到大一直热血追求的,从来没有放弃,看上NBA,迷上了科比,詹姆斯,乔丹,库里,我高中最崇拜的球星是库里,因为他三分实在在准了,投篮速度秒射,基本没人能防的过他,他得分后卫的位置在整场一直保持这种状态,很难有机会能够win过他,那时候我高中就买了他的球衣30号,一直穿着他的球衣去学校打球,觉得自己就跟球星一样,give of confident myself,just do it,那时候一直模仿他投三分,当时我还记得我在学校三分球的投进的几率是80%;跟别人一起打队的时候都是靠三分得分来保持优势,可以说一下课或者放学基本都是跑去操场打球,抢都要抢到位置,一般一打上瘾都是打4,5个小时,都感觉不觉得累, 那种热血澎湃的精神,就想看到一个美女一样,就那种感觉一样,很神奇(如果当时能用在学习上就好了)

    在我高考的前一天,我已经预料到了,我会来创新,创新这地方在等待我,有一种神秘力量在追寻这我,然后高考失败,刚好大专线,之前我为什么选择创新这个大学,是我看这个学校创新(其实一点都不创新),而且学校的环境也是刚建的,觉得环境不错,我就选择了,专业的话,之前非常喜欢玩游戏,所以我就选择了计算机应用技术,之前我以为这个专业是学硬件维修,没想那么多,就是想来学习一下电脑一些配置维修,以后可以多帮助更多人维护电脑,当我来到这个大学后,才发现我们不一样。

    当我步入大学的殿堂,给人感觉就是不一样,首先有一群活泼可爱的舍友,还有一群的志同道合的同学一起为理想和事业做奋斗.在大学的第一堂课是专业介绍课,那时候开着课是曾老师主任来授课讲解,首先当时他先讲解这个专业是要学习什么,然后开始重点的来了,说我们专业主要是学习’PHP’;那时候我就懵逼了,php是什么鬼,听都没听过,高中之前还听过c语言和java语言,就是从来没有听过php,老师那时候讲的,这语言的时候,我完全不能理解,觉得php太深奥了,太神秘了,我这种人最喜欢探究神秘东西,越神秘的东西就越要尝试,那时候老师就留下他的qq群,在加这一刻的群,我在大学里面的生活就改变了,彻底改变了,那时候在群里面,老师就问谁有兴趣加入工作室,只要求三个人,那时候我一听就懵了,’工作室’? 听起来很不错的样子,不过人数要求有点少,而且自己什么都不会,连最基本work文档,excel都不会编辑,很害怕,不过那时候突然自己信心增大,觉得我加入进去可能给我带来很多不一样的收获,会对我未来有所帮助,我那时候就鼓起自己的勇气去找老师面试,第一面遇到老师一点都不紧张,说话语速很稳定,老师那时候也说这个人很开朗,不是很内向的同学,容易和同学互相协同配合的来,我自己也跟老师说我什么都不会,老师那时候也没说什么,也非常鼓励我进去里面学习,当我一进去的时候我内心的非常激动不已,一进去就遇见以为大佬,而且一手神速度的敲代码一大篇,哇塞,根本就看不懂,一脸懵逼,打代码都是php,没先学习到,根本看不懂,就觉得更变魔术一样,很神奇,功能就实现出来了,非常膜拜他,就请教他一下,为我指定道路,那时候我的php之旅就开始了.当我入门php的时候,基础的部分是先听老师讲课的,一开始环境的搭建,echo语句的打印输出,都是从上课老师那里学到了点基础,然后就一直跟着老师那种进度学习,但是发现进度有点慢,学习一直提升不了,曾老师就跟我说,学习编程这东西,不能单单跟老师的进度去学习,这样永远的提升不了,然后老师也介绍一些视频网站给我去看,慕课网等这些网站,从中获取到一些资源,有一点自学目标,刚开始的时候工作室位置还不是很满足,都是每天谁提前去抢着位置学习,那时候是最渴望学习的,但是这样却发现我越是想学越是学习不下,没有固定位置,我一个星期很少去,大部分还是去图书馆和宿舍,这导致了我开始不想去学习了,那时候我们工作室的负责人就开始注意起我了,说你非常不积极,我要把你T了,那时候我非常紧张,非常害怕,说如果离开了,我可能会堕落了,或者不学习.各种可能猜测;到最后我还是留着,是因为之前参加过学校的蓝桥杯选拔赛,成绩有点出众,老师开始提拔我,觉得我还是可以培养的,就开始信任我,在大一的第二学期起,工作室全部焕然一新,所有的办公桌和椅子都全换一整套给我每人每个位置,那时起就开始依赖在工作室学习,每时每刻开始学习编程,不断进度,遇到一个师兄,带我通宵,通宵学习,对php热爱已经逐步加深,越学习越有兴趣,学编程打代码就跟玩游戏,那时刻起,我觉得生活规律都变了,什么都变了,和之前的我完全不同了,php改变了我生活
